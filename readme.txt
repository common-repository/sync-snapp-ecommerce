A wordpress plugin that integrates with Snappfood API.

== Description ==

This plugin uses woocommerce products for their integrity on snappfood platform.

== Installation ==

Install the plugin via wordpress plugin installer or unzip `snapp-ecommerce.zip` to the `/wp-content/plugins/` directory.
